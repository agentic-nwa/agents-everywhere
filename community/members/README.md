# Members

Say hi. Your first commit.

This is the lowest-barrier way to join the repo. Any community member can
add a profile here — drop a short markdown file with who you are, what
you're building, and what you're into. No gate, no review beyond a
maintainer glance to make sure it's a real person.

It's also how the rest of us find people to collaborate with. The more
specific you are about what you're working on, the more likely someone
in the room pings you about it.

## How to add yourself

1. Copy [`TEMPLATE.md`](TEMPLATE.md) to `<your-github-handle>.md` in this
   directory.
2. Fill in the three short sections. Five to ten lines of content is
   plenty.
3. Open a PR. A maintainer will merge it.

If you'd rather have an agent walk you through it, open this repo in
Claude Code / Cursor / Codex / OpenClaw and ask it to help you add a
member profile. It'll use the template.
